# whartonapi

An API to interact with Wharton Webservices


#### auth.py

Provides an API to authenticate a username/password combination

- `auth(username, password)`

    ```
    >>> auth(username, password)
    ```
- `get_token(username, password)`

    ```
    >>> get_token(username, password)
    ABCDEFGHIJK123456
    ```
